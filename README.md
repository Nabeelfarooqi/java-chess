# Java Chess

Object-oriented chess implementation in Java featuring board tile abstraction, piece management, and rule-based game mechanics.

## Features
- Object-oriented architecture
- Board and tile abstraction
- Piece movement logic
- Move validation
- Expandable game design

## Project Structure

src/
 └── com/chess
     ├── board
     ├── pieces
     ├── move
     └── player
